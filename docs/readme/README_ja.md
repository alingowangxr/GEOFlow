# 🚀 GEOFlow: Generative Engine Optimization Flow

[English](./README_en.md) | [繁體中文](../../README.md) | [Español](./README_es.md) | [日本語](./README_ja.md) | [Português](./README_pt_BR.md) | [Русский](./README_ru.md)

**GEOFlow** は、**GEO (生成エンジン最適化)** および **モダン SEO** 向けに設計されたオープンソースのインテリジェント・コンテンツ・エンジニアリング・システムです。

AI 検索エンジン（SearchGPT、Perplexity、Google SGE など）の台頭により、従来のキーワードの詰め込みは効果を失いました。GEOFlow は、体系的なデータ処理、RAG (検索拡張生成) 技術、および意味ネットワークの構築を通じて、生のデータを「高い信頼性」と「構造化された意味」を持つ AI 最適化コンテンツに変換します。

---

## 🔥 なぜ GEOFlow を選ぶのか？

GEOFlow は単なる CMS ではありません。それは **AI コンテンツ工場** です。従来のツールと比較して、「事実に基づく権威性」と「マシンリーダブル（機械可読性）」が大幅に強化されています。

### 主な強力な機能：

#### 1. トップクラスの RAG エンジン：ハイブリッド検索
*   **セマンティック + 高精度：** `pgvector` による意味検索と PostgreSQL `tsvector` による全文検索を同時に組み合わせます。
*   **RRF 融合アルゴリズム：** 産業グレードの RRF (Reciprocal Rank Fusion) アルゴリズムを採用し、AI が取得する参照知識が正確かつ意味的に一貫していることを保証し、AI のハルシネーション（幻覚）を完全に排除します。

#### 2. 事実に基づく権威性：引用・ソース管理システム
*   **自動脚注：** AI がコンテンツ生成時に参照知識に基づいて脚注（例：`[^1]`）を自動的に付与します。
*   **検証可能性：** フロントエンドで双方向ジャンプ機能を備えた「参考文献」リストを自動生成し、Google の E-E-A-T 評価におけるコンテンツの権威性を大幅に向上させます。

#### 3. 意味ネットワーク構築：自動 JSON-LD エンティティ・ラベリング
*   **ナレッジグラフ対応：** 記事から「人物、組織、場所、トピック」エンティティを自動的に抽出します。
*   **Schema.org グラフ：** 複雑な JSON-LD 構造化データを動的に注入し、検索エンジンに対してコンテンツの意味構造を能動的に宣言します。

#### 4. トピッククラスター：セマンティック内部リンクネットワーク
*   **ベクトル関連付け：** ベクトルの類似度に基づいて内部リンクネットワークを自動的に構築します。
*   **権威の継承：** 関連性の高い内部リンクを通じて強力なトピッククラスターを構築し、検索エンジンがあなたのサイトを特定分野の専門家として認識するようにします。

#### 5. 究極のパフォーマンス：インテリジェント・メディア最適化 (Edge Performance)
*   **モダンフォーマット：** 画像を WebP/AVIF 形式に自動変換します。
*   **レスポンシブ・レンダリング：** `<picture>` タグを採用して WebP 優先読み込みを実装し、Core Web Vitals (LCP) 指標を大幅に最適化します。

## 🛠️ 技術アーキテクチャ

GEOFlow は最新のテクノロジースタックに基づいて構築されており、パフォーマンスと拡張性を保証します：

-   **コアフレームワーク:** [Laravel 12](https://laravel.com/) (PHP 8.2+)
-   **データベース:** PostgreSQL + **pgvector** (ベクトルストレージ)
-   **非同期処理:** Redis + **Laravel Horizon** (タスクスケジューリングと監視)
-   **AI 統合:** `laravel/ai` SDK (OpenAI 互換のすべての LLM をサポート)
-   **リアルタイム通信:** Laravel Reverb (WebSocket によるリアルタイムの進捗フィードバック)
-   **フロントエンド・パフォーマンス:** TailwindCSS 4 + Vite

## 📦 クイックスタート

### 環境要件
- PHP 8.2+
- PostgreSQL (`pgvector` 拡張機能がインストールされていること)
- Redis
- Node.js & NPM

### インストール手順
```bash
# 1. リポジトリをクローン
git clone https://github.com/yaojingang/GEOFlow.git && cd GEOFlow

# 2. 自動セットアップスクリプトを実行
composer run setup

# 3. 環境変数を設定
# .env ファイルを編集し、DB_CONNECTION=pgsql および AI_API_KEY を設定
```

### 開発と実行
```bash
# サーバー、キューリスナー、Vite を同時に起動
npm run dev
```

## 📈 ワークフローの概要

1.  **素材のインポート:** PDF、ドキュメント、または URL をアップロードすると、システムが自動的にチャンク化してナレッジベースにベクトル化します。
2.  **タスクの編成:** プロンプト・テンプレート、モデル・ルーティング、および公開間隔を設定します。
3.  **インテリジェント生成:** AI エージェントがハイブリッドなコンテキストを取得し、引用マーカー付きの Markdown コンテンツを生成します。
4.  **セマンティック強化:** エンティティの自動抽出、記事ベクトルの生成、およびセマンティック内部リンクの計算を行います。
5.  **自動配信:** コンテンツは SEO 最適化されたテーマテンプレートを通じて自動的に公開され、動的サイトマップが更新されます。
