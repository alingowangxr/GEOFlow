# 🚀 GEOFlow: Generative Engine Optimization Flow

[English](./README_en.md) | [简体中文](./README_zh_CN.md) | [繁體中文](../../README.md) | [Español](./README_es.md) | [日本語](./README_ja.md) | [Português](./README_pt_BR.md) | [Русский](./README_ru.md)

**GEOFlow** — это интеллектуальная система инженерии контента с открытым исходным кодом, разработанная специально для **GEO (Generative Engine Optimization)** и **современного SEO**.

В эпоху поисковых систем на базе ИИ (таких как SearchGPT, Perplexity, Google SGE) «авторитетность» и «машиночитаемость» контента стали решающими факторами для привлечения трафика. GEOFlow превращает исходные материалы в высококачественный контент, который фактически обоснован, семантически богат и загружается молниеносно благодаря систематической обработке данных, передовой технологии RAG и построению семантических сетей.

## 🌟 Матрица основных функций

### 1. Продвинутый движок RAG (Retrieval-Augmented Generation)
*   **Гибридный поиск:** Сочетает семантический поиск `pgvector` с полнотекстовым поиском PostgreSQL `tsvector` для точного улавливания намерений пользователя и ключевых слов.
*   **Алгоритм слияния RRF:** Использует алгоритм Reciprocal Rank Fusion промышленного уровня для балансировки результатов поиска и полного исключения галлюцинаций ИИ.
*   **Цитирование и поиск источников:** ИИ автоматически расставляет сноски (напр., `[^1]`), а фронтенд генерирует список источников с возможностью перехода, отвечая требованиям Google E-E-A-T.
*   **Интеллектуальная обработка материалов:** Поддержка загрузки PDF/документов и сканирования URL с автоматическим высококачественным разбиением на части и векторизацией.

### 2. Семантическая сеть и GEO-оптимизация
*   **Движок сущностей JSON-LD:** Автоматически извлекает сущности (персоны, организации, места и т. д.) и внедряет структурированные данные Schema.org Graph для взаимодействия с графами знаний поисковых систем.
*   **Семантические внутренние ссылки:** Автоматически строит «тематические кластеры» на основе векторного сходства, обеспечивая высокую релевантность связей между статьями.
*   **Умные сниппеты и Meta:** Автоматически генерирует SEO-совместимые Meta-описания, ключевые слова и фрагменты контента.

### 3. Оркестрация материалов и задач
*   **Централизованное управление:** Встроенные библиотеки заголовков, ключевых слов, изображений и системы управления авторами.
*   **Пакетные задачи ИИ:** Поддержка циклических задач, настройки интервалов публикации и автоматической маршрутизации между моделями.
*   **Поддержка множества моделей:** Совместимость со всеми основными LLM через протокол OpenAI, поддержка стратегий отказоустойчивости (failover).
*   **Управление жизненным циклом:** Охватывает весь процесс от генерации черновиков и ручной/автоматической проверки до публикации по расписанию.

### 4. Техническое SEO и экстремальная производительность
*   **Интеллектуальная оптимизация медиа:** Изображения автоматически конвертируются в формат WebP и отображаются с помощью тега `<picture>` для адаптивной загрузки, что значительно улучшает показатели LCP.
*   **Динамический Sitemap:** Автоматически генерирует XML-карты сайта на основе веса контента для направления поисковых роботов к ключевым материалам.
*   **Мониторинг в реальном времени:** Интеграция с Laravel Horizon для визуального мониторинга очередей задач и отслеживания состояния системы.

## 🛠️ Техническая архитектура

-   **Ядро фреймворка:** [Laravel 12](https://laravel.com/) (PHP 8.2+)
-   **База данных:** PostgreSQL + **pgvector** (Центр векторных данных)
-   **Асинхронная обработка:** Redis + **Laravel Horizon** (Высокопроизводительные очереди задач)
-   **AI SDK:** Официальный пакет `laravel/ai`
-   **Реалтайм:** Laravel Reverb (WebSocket)
-   **Фронтенд:** TailwindCSS 4 + Vite

## 📦 Быстрый старт

### Требования к среде
- PHP 8.2+
- PostgreSQL (с расширением `pgvector`)
- Redis 6.2+
- Node.js и NPM

### Шаги по установке
```bash
# 1. Клонируйте репозиторий
git clone https://github.com/alingowangxr/GEOFlow.git && cd GEOFlow

# 2. Запустите скрипт автоматической настройки
composer run setup

# 3. Настройте переменные окружения
# Отредактируйте .env, установите DB_CONNECTION=pgsql и ваш AI_API_KEY
```

### Запуск разработки
```bash
npm run dev
```
