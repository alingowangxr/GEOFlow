# 🚀 GEOFlow: Generative Engine Optimization Flow

[English](./README_en.md) | [简体中文](./README_zh_CN.md) | [繁體中文](../../README.md) | [Español](./README_es.md) | [日本語](./README_ja.md) | [Português](./README_pt_BR.md) | [Русский](./README_ru.md)

**GEOFlow** — это интеллектуальная система инженерии контента с открытым исходным кодом, разработанная специально для **GEO (Generative Engine Optimization)** и **современного SEO**.

В эпоху поисковых систем на базе ИИ (таких как SearchGPT, Perplexity, Google SGE) традиционное наполнение ключевыми словами потеряло свою эффективность. GEOFlow преобразует необработанные данные в оптимизированный для ИИ контент с «высоким уровнем доверия» и «структурированной семантикой» посредством систематической обработки данных, технологии RAG (Retrieval-Augmented Generation) и построения семантических сетей.

## 🔥 Почему стоит выбрать GEOFlow?

GEOFlow — это больше, чем просто CMS; это **фабрика контента на базе ИИ**. По сравнению с традиционными инструментами, она была глубоко усилена в аспектах «фактологического авторитета» и «машиночитаемости».

### Основные мощные функции:

#### 1. Первоклассный движок RAG: Гибридный поиск
*   **Семантический + Точный:** Сочетает семантический поиск `pgvector` с полнотекстовым поиском PostgreSQL `tsvector`.
*   **Алгоритм слияния RRF:** Использует алгоритм RRF (Reciprocal Rank Fusion) промышленного уровня, чтобы гарантировать, что справочные знания, извлекаемые ИИ, являются точными и семантически связными, полностью исключая галлюцинации ИИ.

#### 2. Фактологический авторитет: Система цитирования и поиска источников
*   **Автоматические сноски:** ИИ автоматически расставляет сноски (например, `[^1]`) на основе справочных знаний во время генерации контента.
*   **Проверяемость:** Фронтенд автоматически генерирует список «Источников» с функцией двустороннего перехода, значительно повышая авторитет контента в рейтингах Google E-E-A-T.

#### 3. Построение семантической сети: Автоматическая маркировка сущностей JSON-LD
*   **Дружелюбность к графу знаний:** Автоматически извлекает сущности «Человек, Организация, Место, Тема» из статей.
*   **Граф Schema.org:** Динамически внедряет сложные структурированные данные JSON-LD для проактивного объявления семантической структуры контента поисковым системам.

#### 4. Тематические кластеры: Семантическая сеть внутренних ссылок
*   **Векторная ассоциация:** Автоматически строит сети внутренних ссылок на основе векторного сходства.
*   **Передача авторитета:** Создает мощные тематические кластеры с помощью внутренних ссылок высокой релевантности, заставляя поисковые системы видеть ваш сайт как эксперта в определенных областях.

#### 5. Экстремальная производительность: Интеллектуальная оптимизация медиа (Edge Performance)
*   **Современные форматы:** Изображения автоматически конвертируются в форматы WebP/AVIF.
*   **Адаптивный рендеринг:** Использует тег `<picture>` для реализации приоритетной загрузки WebP, значительно оптимизируя показатели Core Web Vitals (LCP).

## 🛠️ Техническая архитектура

GEOFlow построен на новейшем технологическом стеке, обеспечивающем производительность и масштабируемость:

-   **Основной фреймворк:** [Laravel 12](https://laravel.com/) (PHP 8.2+)
-   **База данных:** PostgreSQL + **pgvector** (Векторное хранилище)
-   **Асинхронная обработка:** Redis + **Laravel Horizon** (Планирование и мониторинг задач)
-   **Интеграция с ИИ:** SDK `laravel/ai` (Поддерживает все OpenAI-совместимые LLM)
-   **Связь в реальном времени:** Laravel Reverb (Обратная связь о прогрессе в реальном времени через WebSocket)
-   **Производительность фронтенда:** TailwindCSS 4 + Vite

## 📦 Быстрый старт

### Требования к среде
- PHP 8.2+
- PostgreSQL (с установленным расширением `pgvector`)
- Redis
- Node.js и NPM

### Шаги по установке
```bash
# 1. Клонируйте репозиторий
git clone https://github.com/alingowangxr/GEOFlow.git && cd GEOFlow

# 2. Запустите скрипт автоматической настройки
composer run setup

# 3. Настройте переменные окружения
# Отредактируйте файл .env, настройте DB_CONNECTION=pgsql и ваш AI_API_KEY
```

### Разработка и запуск
```bash
# Одновременно запустите сервер, слушатель очередей и Vite
npm run dev
```

## 📈 Обзор рабочего процесса

1.  **Импорт материалов:** Загружайте PDF, документы или URL-адреса; система автоматически разбивает их на фрагменты и векторизует в базу знаний.
2.  **Организация задач:** Настраивайте шаблоны промптов, маршрутизацию моделей и интервалы публикации.
3.  **Интеллектуальная генерация:** ИИ-агенты извлекают гибридные контексты и генерируют Markdown-контент с маркерами цитирования.
4.  **Семантическое улучшение:** Автоматически извлекайте сущности, генерируйте векторы статей и рассчитывайте семантические внутренние ссылки.
5.  **Автоматическое распространение:** Контент автоматически публикуется через оптимизированные для SEO шаблоны тем, а динамические карты сайта (Sitemaps) обновляются.
